# test_task

Инструкция по запуску

1) ставим node v16.20.0 версии
2) из корня директории проекта устанавливаем зависимости npm i
3) в отдельной вкладке терминала из корня директории проекта запускаем json-server командой npm run dev-server
4) в отдельной вкладке терминала из корня директории проекта запускаем сам фронт npm run serve

Решения, которые были приняты:

- классический state manager для vue 2 - vuex
- json сервер с локальной mock DB
- вынос логики api
- глобальные reused стили (приложение маленькое, так обычно использую другое решение, забрал его из предыдущих проектов - расскажу и покажу если интересно - спойлер - функция для упрощения создания БЭМ классов)

При дальнейшем масштабировании можно добавить TS, хотя в таком случае лучше сразу на vue 3 перейти тогда
Далее можно добавить проверку типизации в run-time (zod), тесты и так далее
