Тестовое задание Frontend (Vue)
Стек
Использовать:
Vue.js 2
Vuex
Axios
Можно использовать:
Bootstrap utilities
любой mock API
json-server
статический JSON

Задача
Необходимо реализовать страницу "Заявки партнёров" для CRM.

1. Таблица заявок
Отобразить список заявок в таблице.
Поля:
ID
Имя партнёра
Город
Статус
Дата создания

2. Загрузка данных
Получить данные через API.
Можно использовать:
mock API
json-server
статический JSON
Важно:
запрос должен выполняться через Axios.

3. Поиск
Сделать поиск по:
имени партнёра
городу

4. Фильтр
Добавить фильтр по статусу:
новая
в работе
одобрена
отклонена

5. Сортировка
Добавить сортировку:
по дате создания
по имени партнёра

6. Пагинация
Сделать пагинацию списка.

7. Карточка заявки
При клике на заявку открыть карточку.
Поля карточки:
ID
имя
телефон
город
статус
комментарий
дата создания
Карточку можно реализовать:
через модальное окно
или
отдельный блок на странице

8. Изменение статуса заявки
В карточке заявки должна быть возможность изменить статус.
Доступные статусы:
новая
в работе
одобрена
отклонена
После изменения статус должен:
обновиться в таблице
сохраниться в состоянии приложения
Можно использовать mock API.

9. Валидация формы
Необходимо реализовать базовую валидацию формы.
Обязательные поля:
имя партнёра
телефон
город
Если поле не заполнено:
показать сообщение об ошибке
не сохранять данные

10. Состояния интерфейса
Необходимо обработать:
loading
empty state
error state

11. Оптимизация поиска
Поиск по имени партнёра и городу должен выполняться с задержкой (debounce).
Задержка: 300–500 мс.
Можно реализовать любым способом.
